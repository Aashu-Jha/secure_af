### Functionalities
  - Add credentials.
  - Delete credentials with right swipe.
  - Copy password from outside the tile.
  - View the whole credentials in detail_screen.
  - Day/light theme.
  - Copy accountId/ username, password.
  
### Technology Stack-
 - Flutter
 - Provider
 - SQFlite
 - shared-preferences
